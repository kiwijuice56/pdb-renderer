protein-visualizer

A program to visualize proteins and the function of its individual amino acids through deep learning. Details on the development of this project can be found on this Google document.

Usage

The visualizer accepts a Protein Data Bank (PDB) format file encoding the structure and sequence of a protein. You can download proteins from RCSB Protein Data Bank to test out with this program. Installation and execution requires three steps:

  1. Clone the Protein Visualizer GitHub repository.
  2. Download and configure the pretrained models in protein-visualizer/prose/ according to the instructions on the ProSE GitHub repository.
  3. Navigate to protein-visualizer/ and run python main.py [path to .pdb file].

Before viewing a certain protein chain, the program generates and stores embeddings in protein-visualizer/data/ for future use. Deleting stored embeddings can be a possible fix to errors.
